Para probar este ejemplo deben seguir los siguientes pasos:
1. instalar el wamp server, ejecutarlo y que todos los servicios esten activos
2. copiar esta carpeta a la carpeta WWW que se encuentra en la carpeta donde esta instalado el Wamp server
3. deben crear un usuario y una bd en mysql, y actualizar el archivo "conexion.php" con los datos de su usuario y bd.
   La base de datos se crea ejecutando el script "Base de datos.txt" en la base de datos que crearon desde phpMyAdmin.
